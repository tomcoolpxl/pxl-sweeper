# PXL Sweeper

A modern, high-contrast Minesweeper clone built with Vanilla JS, HTML5, and CSS Grid.

## Quick Start
1. Clone the repository.
2. Open `index.html` in a modern browser (Chromium recommended).

## Features
- Classic Minesweeper gameplay.
- Modern, clean UI.
- Three difficulty levels: Beginner, Intermediate, and Expert.
- Responsive grid layout.

## Tech Stack
- HTML5
- CSS3 (Grid, Flexbox, Variables)
- Vanilla JavaScript
